## 🎮 مميزات اللعبة

### نظام القتال المتقدم
- ⚔️ **هجوم مضاد**: المدافع يرد بـ 50% من قوته
- 🛡️ **منطقة سيطرة (ZOC)**: المرور بجوار عدو يكلف حركة إضافية
- ✂️ **نقاط ضعف**: مشاة>فرسان>مدفعية>دروع>مشاة
- ⭐ **خبرة قتالية**: 5 مستويات مع مكافآت مختلفة
- ⚡ **قدرات خاصة**: كل وحدة لديها قدرة فريدة بتهيئة

### بيئة اللعب الديناميكية
- 🗺️ **خريطة 14×10** سداسية مع 5 أنواع تضاريس
- 🏔️ **ضباب حرب**: لا ترى سوى منطقة وحداتك
- 🌦️ **نظام طقس**: صافي/مطر/ثلج/ضباب/عاصفة - يتغير كل 3 أدوار
- 🏗️ **مباني استراتيجية**: مصنع/مستشفى/حصن/برج مراقبة

### ذكاء اصطناعي متقدم
- 4 مستويات صعوبة: جندي / قائد / استراتيجي / أستاذ الحرب
- AI يحمي مدفعيته ويحاصر ويتراجع الجريح
- AI يستخدم التكتيكات بذكاء حسب الموقف

### لمسات احترافية
- 💾 حفظ تلقائي + 3 خانات يدوية
- 🏆 7 إنجازات قابلة للفتح
- ⏪ تراجع عن آخر حركة
- 📊 مقارنة قتال تفصيلية قبل الهجوم
- 💥 مؤثرات بصرية واهتزاز الشاشة

## 🚀 التشغيل

```bash
# تثبيت المكتبات
bun install

# تشغيل خادم التطوير
bun run dev
```
